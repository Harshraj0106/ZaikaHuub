# 🍽️ ZaikaHub — Food Reels & Discovery Platform

ZaikaHub is a modern MERN stack food discovery platform inspired by short-form reel experiences like Instagram Reels and food delivery apps like Zomato. Users can scroll through engaging food videos, explore dishes, and discover restaurants in an interactive way.

Built using the MERN Stack with React, Redux, Node.js, Express, and MongoDB.

---

# 🚀 Features

* 🎥 Reels-style food video feed
* 🍔 Food discovery interface
* 🔐 User authentication
* 📱 Responsive mobile-friendly UI
* ⚡ Fast frontend using React
* 🗄️ MongoDB database integration
* 🔄 Redux state management
* 🌐 REST API backend
* 📂 Modular project structure

---

# 🛠️ Tech Stack

## Frontend

* React.js
* Redux
* React Router
* Axios
* CSS / Tailwind CSS

## Backend

* Node.js
* Express.js
* MongoDB

# ⚙️ Installation & Setup

## 1️⃣ Clone Repository

```bash
git clone https://github.com/YOUR_USERNAME/zaikahub.git
```

---

## 2️⃣ Install Backend Dependencies

```bash
cd backend
npm install
```

---

## 3️⃣ Run Backend Server

```bash
npm start
```

If `npm start` doesn't work:

```bash
npm run dev
```

---

## 4️⃣ Install Frontend Dependencies

Open another terminal:

```bash
cd frontend
npm install
```

---

## 5️⃣ Run Frontend

```bash
npm run dev
```

---

# 🌐 Open in Browser

After running frontend:

```bash
http://localhost:5173
```

---

# 🔑 Environment Variables

Create a `.env` file inside the backend folder.

Example:

```env
PORT=5000
MONGO_URI=your_mongodb_connection
JWT_SECRET=your_secret_key
```
